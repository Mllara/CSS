CSS de forma básica e simples

Introdução

O CSS (Cascending Style Sheets) descreve como os elementos HTML devem ser exibidos na tela, no papel ou em outras mídias. Ele pode controlar e criar estilos em todos os elementos do HTML, seja de apenas para um elemento ou de vários de uma só vez. O CSS foi desenvolvido em 1996, pelo World Wide Web Consortium, para complementar o HTML, uma vez que este não contém tags para formatação de página, sendo necessária a escrita da marcação para o site. Esse processo se tornou desgastante e caro para desenvolvedores. Por essa razão, uma nova alternativa foi criada. O CSS tem a tarefa de separar o conteúdo do site de sua apresentação visual, alterando elementos como cor do texto, fonte e espaçamento entre blocos, assim como todo o aspecto estético de uma página. Portanto, o HTML é uma linguagem de marcação, ou seja, a base de um site, enquanto o CSS se encarrega de toda a sua apresentação, que deve ser agradável e atrativa para o usuário.

Sintaxe

Seletor: é o elemento de desejamos adicionar estilo, podemos utilizar o próprio nome em si, como sua classe ou identificador único

Declaração: podemos definir quantas declarações forem necessárias e é compreendida com propriedade: valor da propriedade.
Propriedade: nome do estilo que queremos adicionar ao elemento.
Valor da propriedade: é o valor que atribuímos para a propriedade.
Por exemplo:
h1 (Seletor){
font-size(propriedade do Seletor): 24px;(valor da propriedade)
color: red;
height: 100px;
}

Atributo Class

Esse atributo especifica um ou mais nomes de classe para um elemento HTML. Para selecionar elementos com uma classe específica, escreva um carcatere de ponto (.), seguido do nome da classe.

Estilo Inline

Pode ser usado para aplicar um estilo exclusivo para um único elemento. Para usar estilos in-line, adicione o atributo style ao elementorelevante.

Estilo Interno

Uma folha de estilos interna pode ser usada se uma única página tiver um estilo exclusivo. São definidos dentro do elemento style, dentro da seção head de uma página HTML: Por exemplo:
head
style
body{
background-color: branco
} /style
/head

Estilo Externo

Você poderá alterar a aparência de um site em um arquivo com extensão CSS.

Color

Cores são especiicadas usando nomes de cores pré definidos ou valores RGB, HEX entre outros

Background-color

Podemos usar as cores pre definidas par alaterar a cor de fundo da página.

CSS Border

A propriedade CSS border adiciona uma borda ao redor do elemento e corresponde ao agrupamento de outras três propriedades relacionadas:


  • border-width: que corresponde ao tamanho da borda
  • border-style: que define o estilo da borda e é uma informação obrigatória
  • border-color: para definir a cor da borda

Estilos de Fontes

É utilizado a propriedade Font-Style
É usada para especificar o texto em itálico.
Possui 3 valores esta propriedade:

  • normal

    • O texto é mostrado normalmente
  • itálico

    • O texto é mostrado em itálico
  • oblíquo

    • É semelhante ao itálico, mas menos suportado

Tamanho da Fonte

É utilizado a propriedade Font-Size
Permite gerenciar o tamanho do texto:

  • font-size: 16px;
  • font-size: 1em;
  • font-size: 100%;

Tamanho da Fonte Responsivo:

VWé unidade que significa "largura da janela de visualização". Viewport é o tamanho da janela do navegador. 1vw= 1% da largura da janela de visualiação.
Design Responsivo é uma técnica de estruturação HTML e CSS, que consiste em adaptar o site ao browser do usuário sem que seja necessário definir várias folhas de estilos especíificaspara cada resolução, ou seja, é um tipo de design onde o layout fica fluído e variante de acordo com a resolução do usuário.

Alinhamento do texto:

É utilizado a propriedade text-align
